Work Samples

Here are a few web apps developed while learning at Epicodus.

Tamagotchi

A web game developed as an exercise in Javascript objects using jQuery. Try to get Happiness to 10 before Hunger, Tiredness or Loneliness! Github

Simon Says

The classic Simon Says game built here in AngularJS. Click the pattern of lights for as many rounds as you can! Github

Space Pirate

A web game about surviving in a damaged pirate ship in space. This was built in under two class days as an exercise in AngularJS UI Routing. Github

Epifoodus

Epifoodus, AKA the Hangry App, was developed as a team project at Epicodus for those that are too hungry to decide where to eat for lunch. The app gives two random options for a place to eat nearby to Epicodus, and the option to display new choices. Users can create accounts to track what they liked and disliked. This app was built in PHP using Silex, Twig and PostgreSQL. Github
